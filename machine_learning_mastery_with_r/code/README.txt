Machine Learning Mastery With R: Recipes
========================================

Recipes that you can use to bootstrap your machine learning project in R.


About
-----

- Recipes are code snippets not tutorials.
- Recipes provide just enough code to work.
- Recipes are demonstrative not exhaustive.
- Recipes run as-is and produce a result.
- Recipes assume that required packages are installed.
- Recipes use built-in datasets or datasets provided in specific packages.
- Recipes are limited to regression and classification predictive modeling problems.

Usage
-----

1. Find a useful recipe.
2. Copy and paste it into your project.
3. Adapt it to your needs.
